# VueBase

## Sobre

Base com algumas configuraçes para desenvolvimento de projetos.

# Instalação

## Dependências

- [Node.JS](http://nodejs.org)

## Instalação com Yarn
```
yarn install
```

### Inicialização
```
yarn run serve
```

### Build
```
yarn run build
```

### Teste
```
yarn run test
```

### Lint
```
yarn run lint
```
